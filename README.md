## Ferramentas e Serviços utilizados

- Create React App
- GraphQl
- Graphcool
- Apollo
- React Bootstrap
- React Router
- Styled Components
- Sweet Alert

## Comandos para rodando o teste

Os comandos necessários para rodar o projeto em ambiente local são:

 `git clone git@github.com:davidsonsimoes/gdp.git`

 `cd gdp/ && npm install`

 `npm start`
